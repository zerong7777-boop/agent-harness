# Skills Overview

These are the custom workflow skills currently used in this environment.

## research-scout

Use for repo, paper, README, config, and log inspection before code changes.
Output should localize insertion points and recommend the lowest-risk next move.

## module-implementer

Use after the plan is already chosen.
Keep diffs small, preserve the baseline route, and verify immediately.

## experiment-verifier

Use for smoke tests, eval runs, metric checks, and blocker classification.
Never claim completion without command-backed evidence.

## memory-and-handoff

Use to write durable `STATUS.md`, `VERIFY.md`, `NEXT.md`, and milestone notes.
Keep facts separate from recommendations.
